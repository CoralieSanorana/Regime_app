# Projet: Appplication de sante

## Front office (analyse du sujet)
- Inscription
    - informations personnelles + contact
        (nom, prenom,genre, email, adresse)
    - informations de santes 
        (poids, taille)
    - securite
        (pwd)

- page login
    - email + pwd

- Profil de l'utilisateur 
    - Informations personnelles 
        - Nom, prénom, date de naissance, email,adresse 
    - Informations de santé 
        - Genre, poids actuel, taille
    - Affiche son indice de masse corporelle(IMC) 

- choix d'un objectif
    - augmenter poids
        (saisir le poids cible)
    - perdre poids
        (saisir le poids cible)
    - atteindre IMC ideal

- suggestion des regimes et activites
    - trouver le poids a combler
    - afficher les activites sportives adequates
    - voir les regimes  adequat capablent de combler ce poids
        - details du regime (Nom, % de nutriments, effet par jour), duree, prix par jour, prix total
    - l'utilisateur choisi le regime qui lui convient
    - export pdf

- porte monnaie
    - bouton 'Recharger mon porte-monnaie'
    - saisi du code de recharge
    - validation du transaction

- option Gold + remise de 15% sur le prix des regimes
